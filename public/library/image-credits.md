## License posture (applies to all entries)

Every portrait below was published before 1929 in the United States, putting each unambiguously in the public domain. No permissions or payments are required for use. Attribution is provided as a courtesy to readers and to the institutional sources that host the scans, not as a legal obligation.

## Maintenance

Update this file when:
- A new portrait replaces an existing one (note source URL and date of swap).
- A new author entry is added to the Library page (Wilson, Audubon, Coues, etc., for v2 expansion).
- An institutional steward (Connecticut Audubon, Birdsacre, Sherman Swift Tower, Princeton Mudd Library) confirms permission for an additional image.
- A higher-resolution PD scan of an existing portrait surfaces (especially Margaret Morse Nice and Neltje Blanchan, currently the lowest-resolution sources).
